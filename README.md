# Real Estate CRM — Kenya
A WhatsApp-first CRM built for Kenyan real estate agents.
Built with Django REST Framework + React + PostgreSQL + AI Automation.

## Status
🔨 Day 1 — Active Development

## Stack
- Backend: Django + Django REST Framework
- Frontend: React + Redux
- Database: PostgreSQL
- Async: Celery + Redis
- AI: Groq API(llama 3)
- Deployment: Railway (backend) + Vercel (frontend)

## The Problem
Kenyan real estate agents manage leads on WhatsApp and track
properties on Excel. This system fixes that.
